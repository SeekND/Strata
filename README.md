# Strata
Know before you dig.

Strata is a Star Citizen mining reference tool — find the best locations 
for any ore, compare refinery yields across stations, and plan multi-stop 
mining runs to maximize every trip.

Data sourced from Regolith Survey Corps and UEX Corp.
